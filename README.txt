# 🌍 Geo Reporting Platform

> **Plataforma de geolocalização para cidadania ativa e monitoramento urbano.**

![React](https://img.shields.io/badge/React-20232A?style=for-the-badge&logo=react&logoColor=61DAFB) ![TailwindCSS](https://img.shields.io/badge/Tailwind_CSS-38B2AC?style=for-the-badge&logo=tailwind-css&logoColor=white) ![TypeScript](https://img.shields.io/badge/TypeScript-007ACC?style=for-the-badge&logo=typescript&logoColor=white) ![Vercel](https://img.shields.io/badge/Vercel-000000?style=for-the-badge&logo=vercel&logoColor=white)

O **Geo Reporting Platform** é uma interface moderna desenvolvida para conectar cidadãos à gestão pública. Através de um mapa interativo, usuários podem reportar incidentes como buracos, falta de iluminação ou problemas de saneamento.

## 🎯 Objetivo do Projeto

Este projeto demonstra o domínio de tecnologias **Frontend** e a capacidade de transformar dados geográficos em uma interface intuitiva.

* **Dificuldade de Comunicação:** Centraliza as reclamações em um canal visual.
* **Transparência:** Permite visualização pública dos pontos reportados.
* **Engajamento:** Interface rápida e responsiva para participação popular.

---

## ✨ Diferenciais Técnicos

* 📍 **Integração de Mapas:** Camadas interativas com Leaflet.js.
* 🧩 **Componentização:** Código modular e reutilizável.
* 📱 **Mobile First:** Totalmente responsivo para uso em campo.
* ⚡ **State Management:** Fluxo de dados otimizado.
